# FPSEnhancer

一款专为《无畏契约 (VALORANT)》设计的游戏分辨率工具，通过自动监听游戏进程，在启动时应用临时4:3分辨率，退出后自动恢复系统设置。

## 功能特性 ✨

- 🎮 **自动进程监听**：实时检测游戏进程的启动与退出，自动触发配置切换
- 🖥️ **4:3 分辨率自动切换**：游戏启动时自动切换到自定义 4:3 拉伸分辨率，退出后恢复原生分辨率
- ⚡ **一键启动游戏**：配置完成后，可直接从工具内一键启动游戏
- 🔧 **灵活参数配置**：支持自定义分辨率、帧率、进程名等核心参数
- 🎯 **无畏契约深度适配**：自动扫描游戏安装路径，一键应用优化预设

## 使用前提 📋

1.  **管理员权限运行**：修改系统分辨率需要管理员权限，请右键选择“以管理员身份运行”。
2.  **添加自定义分辨率**：在 NVIDIA 控制面板中添加 4:3 分辨率（如 1440×1080 或 1568×1080）。
3.  **设置缩放模式**：NVIDIA 控制面板 → 调整桌面尺寸和位置 → 选择“全屏” → 勾选“GPU 执行缩放” → 勾选“覆盖程序设置”。

## 快速开始 🚀

### 1. 下载安装

前往项目 [Releases](https://github.com/Garusuta/fps_enhancer/releases) 页面，下载最新版本的安装包或可执行文件。

### 2. 配置游戏路径

- **方式一（手动选择）**：在「通用」设置页，点击「选择游戏路径」，手动定位到 `VALORANT.exe` 或游戏启动器。
- **方式二（自动扫描）**：先启动《无畏契约》，然后在「无畏契约」专属页点击「扫描游戏路径」，工具将自动检测并填充路径。

### 3. 启动进程监听

在「通用」设置页，打开“进程监听器”开关。工具将开始后台监听，等待游戏进程启动。

### 4. 开始游戏

- 点击工具内的「一键启动」按钮，或直接从游戏客户端启动。
- 分辨率将在游戏窗口出现时自动切换为 4:3 拉伸模式，游戏退出后自动恢复。

## 界面功能说明 🛠️

- **修改CFG文件**：修改无畏契约国服的CFG文件，使其强制应用预设的分辨率。
- **显示任务栏**：如果无畏契约窗口有遮挡，请打开显示任务栏。
- **调整画质专区**：进入专门的画质设置面板，对游戏内的图形选项进行预设和调整。
