# FPSEnhancer

游戏帧率优化工具，自动监听游戏进程，启动时应用高性能配置，退出后自动恢复。

专为无畏契约 (VALORANT) 优化，支持 4:3 拉伸分辨率自动切换。

## 功能特性

- 🎮 **自动监听** - 检测游戏进程启动/退出，自动切换配置
- 🖥️ **分辨率切换** - 游戏启动切 4:3，退出自动恢复
- ⚡ **一键启动** - 配置好路径后，一键启动游戏
- 🔧 **灵活配置** - 支持自定义分辨率、帧率等参数
- 🎯 **无畏契约适配** - 自动扫描游戏路径，一键应用预设

## 使用前提

1. **管理员权限运行** - 修改分辨率需要管理员权限
2. **添加自定义分辨率** - 在 NVIDIA 控制面板中添加 4:3 分辨率（如 1440x1080 或 1568x1080）
3. **缩放模式设置** - NVIDIA 控制面板 → 调整桌面尺寸和位置 → 全屏 → GPU 执行缩放 → 覆盖程序设置

## 快速开始

### 1. 下载安装

前往 [Releases](https://github.com/Garusuta/fps_enhancer/releases) 下载最新版本。

### 2. 配置游戏路径

- **方式一**：在「通用」页点击「选择游戏路径」手动选择
- **方式二**：启动游戏后，在「无畏契约」页点击「扫描游戏路径」自动检测

### 3. 启动监听

在「通用」页打开监听器开关，程序会自动监听游戏进程。

### 4. 开始游戏

点击「一键启动」或直接从游戏客户端启动，分辨率会自动切换。
